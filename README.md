## Running Options

1. **Local Environment (default):**

   ```bash
   python main.py
   ```

2. **UAT Environment:**

   ```bash
   python main.py uat
   ```

3. **Production Environment:**

   ```bash
   python main.py prod
   ```

The environment parameter determines which database configuration to use (local, UAT, or production). If no parameter is provided, it defaults to "local".
